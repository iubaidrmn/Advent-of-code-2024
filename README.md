## About Advent of Code

[Advent of Code](https://adventofcode.com/) is an annual event of programming puzzles that runs from December 1st to December 25th. Each day unlocks a new challenge, blending storytelling with problem-solving. It's a great way to:

- Learn new programming skills.
- Compete with friends.
- Dive into a festive coding tradition!

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/advent-of-code-2024.git
   cd advent-of-code-2024
   ```

2. Navigate to the folder for a specific day and run the solution:

   ```bash
   cd day01
   python part1.py
   python part2.py
   ```

3. Update the repository daily with new solutions.
